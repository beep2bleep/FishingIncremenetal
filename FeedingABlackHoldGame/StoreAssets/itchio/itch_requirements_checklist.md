itch.io Page Checklist
Source: itch.io creator quality guidelines + page design docs

Core page elements to prepare:
- Cover image and clear title presentation
- Short summary at the top of page
- Long description with readable sections
- Real gameplay screenshots (multiple)
- Optional trailer or gameplay GIF near top
- Accurate metadata (genre, tags, platforms)

Recommended content order for conversion:
1) One-sentence hook
2) 3 to 6 key features
3) Screenshot strip or embedded trailer
4) Progression / systems explanation
5) Controls and session style
6) Build notes / update cadence

Image guidance:
- itch.io docs emphasize quality and clarity over one strict dimension set
- Use high-resolution landscape screenshots (1920x1080 preferred)
- Keep key text inside safe center area for responsive layouts

Metadata guidance:
- Category: Game
- Genre: Strategy / Simulation (or as appropriate for your release taxonomy)
- Tags: Auto Battler, Idle, Incremental, Strategy, Singleplayer
- Platforms: list only platforms you actively ship

Quality bar:
- No placeholder lorem text
- No misleading screenshots
- Description must match currently playable systems
